# Technology Stack

## Frontend
- **Framework:** Next.js (App Router)
- **UI Library:** Material UI (MUI)
- **Styling:** Custom Glassmorphism design system
- **State Management:** Zustand
- **Diagramming:** React Flow (@xyflow/react)

## Backend
- **Framework:** FastAPI (Python)
- **Database:** PostgreSQL
- **Migrations:** Alembic
- **API Style:** REST

## Engineering & Shared Logic
- **Hydraulic Engine:** Python (FastAPI integration)
- **Shared Packages:** TypeScript monorepo packages (`ui-kit`, `physics`, `shared`) managed via Turborepo.

## Infrastructure
- **Containerization:** Docker & Docker Compose
- **Process Management:** Supervisord (for production images)
- **Environment:** Node.js (v18+) and Python (3.11+)

## Development Workflow
- **Languages:** TypeScript, Python
- **Monorepo Tool:** Turborepo
- **Package Manager:** npm
- **Linting/Formatting:** ESLint, Prettier
